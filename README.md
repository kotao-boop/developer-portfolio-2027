# 2027 Interactive Developer Portfolio

A futuristic, interactive developer portfolio template designed for the year 2027. It completely breaks the conventional "laws of digital gravity" by incorporating advanced Canvas-based particle physics, a terminal typing effect, and an ultra-modern Bento Grid layout.

## 🔗 Live Demo
[View Live on GitHub Pages](https://kotao-boop.github.io/developer-portfolio-2027/)

## ⚡ Features
- **Canvas Particle Engine**: A background that reacts dynamically to user mouse movements using native Canvas API physics.
- **Terminal Typing Effect**: A classic retro-futuristic command-line interface that executes commands asynchronously.
- **Bento Grid Architecture**: A responsive, glassmorphism-inspired grid system ideal for showcasing high-end developer projects.
- **Glitch Aesthetics**: Custom CSS keyframes producing an authentic neon glitch effect for hero text.

## 🛠 Technology Stack
- Vanilla HTML5
- Custom CSS (No external frameworks, full control over animations)
- Vanilla JavaScript (ES6+ Modules)

## 📦 Usage
Simply clone this repository and open `index.html` in your browser. No build steps or package managers are required. It is completely standalone and optimized for immediate deployment.

```bash
git clone <repository-url>
cd developer-portfolio-2027
# Open index.html
```
